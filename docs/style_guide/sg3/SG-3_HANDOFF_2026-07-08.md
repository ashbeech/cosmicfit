## 5. What was fixed and independently verified before this run

An external second-pass audit verified all of these **in code and by running
them** (do not re-litigate; do not undo):

- **Resume safety:** `backfill_narratives.existing_passing_sections` re-gates
  every existing section against the current gate on `--resume-from-partial`
  and drops failures. Verified: against the run-1 cache it kept only
  1,828/2,384 sections and would skip 0/149 clusters as complete.
- **Pass-over vs leak-gate conflict (88/192 clusters):**
  `sg_validation.find_literal_leaks(text, section, allowed_phrases)` strips
  the chart's own pass-over list before lexicon matching. Verified: naming
  pass-over colours passes; a stray "camel"/"oxblood" still blocks.
- **Model pinning:** `MultiKeyGenerator._new_client` disables
  `GeminiClient`'s fallback (a retired model is a fatal error, never a silent
  downgrade to flash) and asserts the constructed model matches. The model id
  is written on every run-log entry.
- **Intro + closing gating:** `gate_intro` (dash/tic/leak/spelling/season)
  folds into `gate_section`; `gate_closing` also checks spelling, season
  words, and literal colours (pass-over-aware). Verified live with crafted
  inputs.
- **Length hard block:** 200 words default, 280 for `style_core`
  (`sg_generate._LENGTH_BLOCK`). Verified: blocks at 201.
- **Audit tool raised to gate coverage:** leak check over ALL Group-B
  sections via the same gate function, case-insensitive formula checks,
  per-register stamping (≥60% of a register), intro hygiene checks.
- **Holistic pass re-gates with the example-copy guard and intro.**
- Profile parity 16/16 and the gate self-test pass on current code.

**Accepted risks, documented in `SG-3_HANDOFF.md` §9a (do not change without
cause):** metal/stone names are deliberately not leak-gated (false-positive
risk: "steel blue" is a colour, "gold" is common); required-phrase recurrence

no-op for single-entry pools (hardware has 1 test/1 trap per register — check
the re-run audit's hardware stamping before expanding the library); the
`texture_bad` fibre lexicon gap is flagged for follow-up (the
required-placeholder check still forces `{texture_bad_*}`).

**Known cosmetic nits (harmless, fix only if touching the file anyway):** the

  pass 16/16.

  `data/style_guide/style_guide_rules.json` (single source of truth, also for

  parallel workers, re-gating resume, run log, quarantine/triage sidecars.

- `.env` — `GEMINI_MODEL=gemini-3.1-pro-preview`, `GEMINI_API_KEY_1..3`.
- Golden guides: `docs/style_guide/golden/*.md`; rubric:
  `docs/style_guide/style_standard.md`; frozen contracts:
  `docs/style_guide/decisions/injection_contract_freeze.md`.

  shipped app still uses the 576-cluster v1 cache
  (`data/style_guide/blueprint_narrative_cache.json`) — untouched. Scaling to
  576 is SG-4.
